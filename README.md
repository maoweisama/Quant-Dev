# Quant-Dev: 豆粕期货量化策略研究

## 项目背景
本项目独立完成豆粕期货的量化策略研究与实现，涵盖数据获取、清洗、因子设计和交易信号生成。目标是建立可回测的策略框架，验证因子在历史数据中的有效性，为策略优化提供基础。

## 方法与流程
1. **数据获取与清洗**  
   - 收集 2006–2025 年豆粕期货历史行情  
   - 异常值处理、缺失值填充及结构化转换  
2. **因子工程**  
   - 构建自定义因子（如价格动量、波动率指标）  
   - 标准化处理并进行特征衍生  
3. **信号生成与回测**  
   - 基于因子逻辑生成交易信号  
   - 执行固定止损/止盈规则回测  
   - 计算策略累积收益、最大回撤、胜率等指标

## 回测结果
- 策略在历史数据中实现正向收益  
- 最大回撤控制在可接受范围  
- 胜率及盈亏比表现稳定  

## 项目亮点
- 完整策略研究流程实现（数据 → 因子 → 信号 → 回测）  
- 全流程独立完成，便于快速迭代和扩展到其他品种  
- 代码结构清晰，可直接用于策略验证或研究
